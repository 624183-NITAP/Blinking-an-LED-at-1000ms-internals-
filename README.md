# Blinking-an-LED-at-1000ms-internals-

*A simple embedded systems project that demonstrates LED blinking on an Arduino Nano with ON and OFF durations as close to **1000 milliseconds** as possible.*

## Project Overview

This project implements the classic "Blinky" application used in embedded systems development to verify:

* Microcontroller operation
* GPIO configuration
* Timing accuracy
* Hardware interfacing
* Program upload and execution

The LED is driven with a period of approximately 2 seconds:

* ON Time ≈ 1000 ms
* OFF Time ≈ 1000 ms


## Learning Outcomes

* Embedded C/C++ Programming
* GPIO Control
* Timing Functions
* Breadboard Prototyping
* Microcontroller Fundamentals
